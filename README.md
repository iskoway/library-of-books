# Library of Books

Веб-сайт библиотеки книг на `Django` с использованием стандартных шаблонов Django Templates.  
Проект реализует каталог книг с современным адаптивным интерфейсом, поиском, фильтрацией по категориям и формой добавления новых книг.

## Возможности

- просмотр главной страницы с подборкой книг
- каталог всех книг
- поиск по названию, описанию, автору и категории
- список книг по категориям
- страница детальной информации о книге
- добавление новой книги через форму
- админ-панель Django для управления данными
- адаптивная верстка для десктопа и мобильных устройств

## Модели

### `Author`

- `full_name` — имя автора
- `biography` — краткая биография

### `Category`

- `name` — название категории
- `slug` — уникальный slug для URL
- `description` — описание категории

### `Book`

- `title` — название книги
- `author` — связь с автором
- `category` — связь с категорией
- `published_year` — год издания
- `description` — описание книги
- `created_at` — дата добавления

## Установка и запуск

### 1. Клонирование проекта

```bash
git clone <repo-url>
cd library-of-books
```

### 2. Создание виртуального окружения

```bash
python3 -m venv .venv
```

### 3. Активация виртуального окружения

macOS / Linux:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

### 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 5. Применение миграций

```bash
python manage.py migrate
```

### 6. Создание суперпользователя

```bash
python manage.py createsuperuser
```

### 7. Запуск сервера

```bash
python manage.py runserver
```

После запуска проект будет доступен по адресу:

```text
http://127.0.0.1:8000/
```

## Основные маршруты

- `/` — главная страница
- `/books/` — список всех книг
- `/books/add/` — добавление новой книги
- `/books/<id>/` — страница книги
- `/categories/<slug>/` — книги выбранной категории
- `/admin/` — административная панель
